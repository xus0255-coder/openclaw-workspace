### 技术踩坑清单（关键学习）
| 问题 | 根因 | 解决方案 |
|------|------|----------|
| 无画面 | 横屏图(720×480)拉到竖屏(720×1280)变形 | Unsplash `?w=1080&h=1920&fit=crop` 获取竖版图 |
| 字幕消失 | ASS字体6号=6像素，不可见 | 中文6号≈7.5pt，ASS应设FontSize=28 |
| 字幕渲染失败 | Shell编码问题：`&H`被当管道符、中文乱码、`$()`被PS解析 | **ASS文件格式**直接让libass读文件，跳过shell |
| 字幕不流动 | drawtext单文本无分段 | ASS事件逐句时间轴，27段分段 |
| 僵尸进程 | 旧cmd残留一直跑`_check.mp4` | `taskkill /f /im cmd.exe` 杀进程 |
| batch编码 | write工具写UTF-16 batch，cmd不识别 | Python的`open().write()`写ANSI batch |
| 时长不足 | fade过渡参数不正确，只渲染27s | `setpts=PTS-STARTPTS+26/TB` 延迟第二图播放时间 |
